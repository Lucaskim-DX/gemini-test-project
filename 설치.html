Gemini CLI 설치 및 종료 가이드

Gemini CLI는 구글이 공개한 오픈소스 AI 에이전트로, 터미널에서 강력한 Gemini AI 기능을 무료로 사용할 수 있게 해주는 명령줄 도구입니다.

1. 설치 전 필수 준비

  • Node.js 18 버전 이상이 반드시 설치되어 있어야 합니다.
    Node.js가 없다면 공식 Node.js 다운로드 페이지에서 운영체제에 맞게 설치하세요.
  • 구글 계정(개인/Workspace 모두 가능)
  • 인터넷 연결

2. Gemini CLI 설치 방법

가장 쉬운 방법: npx로 바로 실행

npx https://github.com/google-gemini/gemini-cli
  • 설치 없이 바로 실행할 수 있습니다.
  • 처음 실행 시 구글 계정 인증(브라우저 연동) 후 사용 가능합니다.

전역 설치(권장)

npm install -g @google/gemini-cli
gemini
  • 한 번만 설치하면, gemini 명령어로 언제든지 터미널에서 바로 실행할 수 있습니다.
  • 설치가 끝나면, 첫 실행 시 테마 선택 및 구글 계정 인증 절차가 진행됩니다.
설치가 완료되면, 터미널에 gemini를 입력해 Gemini CLI를 시작하세요.

3. 인증 및 고급 설정

  • 개인 사용자는 구글 계정으로 로그인만 하면 무료로 분당 60회, 일일 1,000회 요청이 가능합니다.
  • 더 높은 한도나 기업용 기능이 필요하면 Google AI Studio에서 API 키를 발급받아 환경 변수로 등록하세요.
    export GEMINI_API_KEY="YOUR_API_KEY"

4. 종료 방법

  • 가장 권장: /quit 명령어 입력
  • 또는: Ctrl + C 키를 눌러 종료
  • 터미널 창을 직접 닫아도 종료되지만, /quit 명령어를 사용하면 세션 통계(요청 횟수, 토큰 사용량 등)를 확인할 수 있습니다.
종료 시 세션 요약(요청 횟수, 토큰 사용량, 소요 시간 등)이 자동으로 표시됩니다.

5. 참고 및 문제 해결

  • 설치 과정에서 문제가 발생하면 node -v로 Node.js 버전을 확인하세요. 18 이상이어야 합니다.
  • 인증 오류 시, 구글 계정 및 네트워크 상태를 점검하세요.
  • Workspace 계정은 추가 환경변수 설정이 필요할 수 있습니다.
  • 공식 문서 및 gemini --help 명령어로 도움말을 확인하세요.

6. 주요 명령어 요약

  • /clear : 화면/컨텍스트 초기화
  • /chat save <태그> : 대화 저장
  • /chat resume <태그> : 저장된 대화 불러오기
  • /quit : 안전하게 종료
